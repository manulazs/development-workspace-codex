# Rejected Patterns

Use this directory for workflows that were considered but should not be reused without new evidence.

## Entry Template

```markdown
# Rejected Pattern: <name>

## Status

- Rejected date:
- Reviewer:
- Replacement:
- Review again after:

## Context

Why this pattern looked useful.

## Rejection Reason

The evidence or risk that made it unsuitable.

## Do Instead

Approved alternative workflow.

## Risk If Revived

What can break if this pattern returns without review.
```
